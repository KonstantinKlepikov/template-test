# Agent Instructions: py-project-template

## Overview

Python project using Poetry, Docker, pytest, mypy, flake8, and ruff.
Development runs inside Docker — commands are executed via `make` targets.

## Key Commands

| Task                                             | Command                                               |
|--------------------------------------------------|-------------------------------------------------------|
| Start dev environment                            | `make serve`                                          |
| Stop                                             | `make down`                                           |
| Run checks (tests + lint + types)                | `make check`                                          |
| Show stack config                                | `make config`                                         |
| Create a python file with optional prompt header | `make create --path=src/module.py [--pr=prompt_name]` |
| Install deps locally                             | `poetry install --with dev --no-root`                 |

`make check` runs inside the container: `pytest; mypy src; mypy tests; flake8 src; flake8 tests`

## Project Layout

- `src/` — application source (mounted into container at `/src`)
- `tests/` — pytest tests (mounted at `/tests`)
- `scripts/` — shell scripts used inside the container
- `docker-compose/` — Docker configs; `docker-stack.yml` is generated by `up-dev.sh`, not edited manually
- `research/` — scratch/experiment files, not production code
- `prompts/` - system prompts of project

## Conventions

- **Python 3.12+**, type annotations required; mypy runs with `check_untyped_defs=true`
- **Line length:** 88 (ruff, flake8, blue formatter)
- **Quotes:** single (ruff + flake8-quotes enforce this)
- **Imports:** sorted via isort with `black` profile; `flake8-isort` enforces order
- **Complexity:** max cyclomatic complexity 10 (flake8)
- **Async tests:** `pytest-asyncio` with `asyncio_mode = auto`
- **No warnings** in test output (`-p no:warnings` flag)

## Tooling Notes

- `ruff` is available for fast linting/formatting but `flake8` is the enforced linter in CI (`make check`)
- `radon` is available for complexity metrics: `radon cc src -s`
- `mypy` paths are `src` and `tests` (configured in `pyproject.toml`)
- `blue` is the formatter (Black-compatible, single quotes, preview mode)
- `docker-stack.yml` is auto-generated — do not commit manual edits to it
